## <b>MbedTLS_HW_ALT Application Description</b>

The NUCLEO-H553ZG MbedTLS_HW_ALT project is based on a recursive copy of STM32H5F5J-DK/Applications/MbedTLS_HW_ALT/ content and few changes.

### <b>How to use it ?</b>

  - After the copy from STM32H5F5J-DK/Applications/MbedTLS_HW_ALT/, update the project setup for your favorite IDE of the example your are looking for (change target to H553ZG).
  - Follow the readme of the copied example.
