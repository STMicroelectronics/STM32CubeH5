## <b>MbedTLS_SW Application Description</b>

The NUCLEO-H553ZG MbedTLS_SW project is based on a recursive copy of STM32H5F5J-DK/Applications/MbedTLS_SW/ content and few changes.

### <b>How to use it ?</b>

  - After the copy from STM32H5F5J-DK/Applications/MbedTLS_SW/, update the project setup for your favorite IDE of the example your are looking for (change target to H553ZG).
  - Follow the readme for the copied example.
